# Truck Logistics Route Optimization Using A* Search

**Module:** Artificial Intelligence (CMPG 313)
**Practical:** Lab 5
**Institution:** North-West University

## Description

This project implements the A* Search Algorithm to find the optimal delivery route for a logistics truck across the Mahikeng Local Municipality road network. Starting from Disaneng and targeting Coligny as the goal, the algorithm minimizes travel cost by combining actual road distances (km) with traffic-based heuristic values per city.

The code was provided incomplete and with bugs as part of the practical assessment. All issues were identified and corrected before execution.

## Files

- `A_star_search.py` — A* algorithm implementation
- `A_star_search_testcase.py` — Truck journey simulation

## How to Run

python A_star_search.py
python A_star_search_testcase.py

## Output

Optimal Path:  Disaneng -> Mahikeng -> Bakerville -> Lichtenburg -> Coligny
Total Cost:    110 km
